Première utilisation

  • Créer et publier des produits pour la première fois

    • Solution

      Ce tutoriel va vous guider pour créer et publier vos produits.


      Pour publier vos produits vous aurez à effectuer les 5 étapes suivantes :

      1. Créer des produits dans l'interface
      2. Classer ces produits dans les catégories disponibles
      3. Compléter les attributs
      4. Ajouter des images
      5. Publier vos produits lorsque vous le souhaitez


      A savoir : tant que vous ne faites pas l'action de publication vos produits ne peuvent pas être visibles par votre distributeur. Tout ce que vous faites sur cette plateforme vous est entièrement confidentiel.

      Pour chacune des étapes ci-dessus vous pouvez utiliser l'assistant pour vous aider à vous guider. Pour la première étape, la création de produit dans l'interface, il existe différentes méthodes de création de produit en fonction de votre besoin.


      Cliquez sur le bouton "Revenir à l'assistant" en haut à droite et sélectionnez le tutoriel Créer des produits dans la section "Se former" puis faites de même pour les autres étapes.


      Très bonne publication.


Se former

  • Créer des produits
    • Quelle méthode souhaitez-vous utiliser?
      • Je veux créer des produits directement dans l'interface

        • Solution

          Cette solution vous permet de créer vos produits directement dans l'interface.

          Lisez le tutoriel en entier avant de commencer.


          Etape 1 : Créer les références

          1. Dans le menu supérieur, cliquez sur Classification
          2. Cliquez sur le bouton bleu Actions
          3. Dans le menu qui s'ouvre cliquez sur Créer des produits
          4. Dans l'interface qui s'ouvre, éditer un produit par ligne. Vous pouvez coller une liste de référence depuis Excel.
          5. Enfin cliquez sur le bouton Créer les produits en bas à droite.

          Etape 2 : Classer les produits

          1. Assurez vous d'être toujours sur le menu Classification
          2. Sélectionnez ensuite des produits et classés les dans le panneau de droite par glissé/déposé (comme avec des fichiers sur votre ordinateur)

          Etape 3 : Compléter les attributs

          1. Dans le menu supérieur, cliquez sur Attributs
          2. Compléter l'ensemble des attributs. Chaque attribut obligatoire est indiqué en rouge.

          Important!

          Pour afficher les attributs spécifiques à une catégorie vous devez au préalable sélectionner une catégorie de plus bas niveau.

          Pour afficher le menu des catégories dans la vue Attributs, cliquez sur le bouton en haut à gauche.

          Etape 4 : Ajouter les images

          1. Dans le menu supérieur, cliquez sur Images
          2. Sur chaque produit déposez un ou des fichiers dans le panneau de droite. A ce stade l'image est associée au produit mais pas à la vue.
          3. Lorsque vous voyez la miniature de l'image que vous venez de déposer, il faut que vous la glissiez/déposé dans la vue dans laquelle vous souhaitez l'associer.

          Etape 5 : Publiez les produits

          1. Sélectionnez une liste de produit
          2. Cliquez sur le bouton bleu Actions
          3. Cliquez ensuite sur Publier les produits

          Félicitations!

          Vous venez de publier vos premiers produits

          N'hésitez pas à consulter les autres tutoriaux pour gagner plus de temps

          Questions posées fréquement

          J'ai un message d'erreur de publication

          Vérifier dans l'ordre les points suivants et recommencez :

          1. Est-ce que vos codes EAN 13 sont valides?
          2. Est-ce que vos produits sous classés dans une catégorie de dernier niveau?
          3. Est-ce que vous avez complété tous les attributs obligatoires?
          4. Est-ce que vous avez vérifié que vos produits n'auraient pas des attributs obligatoires non renseignés?
          5. Est-ce que vous avez complété toutes les images obligatoires?

          Etape 3

          Une fois votre saisie terminée, vous devez réimporter le fichier dans l'interface. Pour ce faire :

  • Classer des produits

    • Tutoriel

      Pour classer vos produits cliquez dans le menu central sur le bouton :




      Vous pouvez sélectionner et classer des produits comme vous le faites quand vous déplacez des fichiers sur votre ordinateur. La classification se fait dans le menu de droite.

      Vous pouvez sélectionner plusieurs produits d'affilé en utilisant la touche SHIFT + clic de souris, ou en utilisant CTRL + clic de souris, comme pour sélectionner des fichiers sur votre ordinateur.
